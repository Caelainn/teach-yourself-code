Goal: To give you enough tools in one hour to teach yourself how to code.

It will actually take months, even years, to get good at these things. Start small, stay patient and keep pushing yourself to learn new things.

Where to start

It doesn't really matter what language you learn first. What's important is learning how to think like a developer. Sign up for GitHub and start browsing others' code. Learn how to maneuver around your computer on the command line, and then how to setup a Github repository. Start keeping track of your projects, however small, as you build them by committing your changes to Github.

Set up your computer

Assuming you're on a Mac, which is all I'm qualified to teach you on, you'll need to have a text editor. Recommended: Sublime Text 3 or Atom. Your computer will also tell you to download XCode.
